# Максим

Python-разработчик из Санкт-Петербурга. Специализируюсь на создании Telegram-ботов с интеграцией AI, платёжных систем и компьютерного зрения. В портфолио — 9 завершённых проектов: от учебных платформ на базе нейросетей до ботов для продажи VPN-доступа с автоматической выдачей конфигов.

Умею работать с асинхронным Python (`asyncio`, `aiogram`, `python-telegram-bot`), подключать AI-модели (OpenAI, DeepSeek, gpt4free), обрабатывать документы и изображения, интегрировать платёжные шлюзы. Пишу читаемый код, использую Git, работаю на Linux.

Открыт к позиции **Junior Python Developer** — готов учиться, брать реальные задачи и расти в команде.
